This is not meant to be executed in a web browser :(

Please use the ant script build.xml provided in the base directory.

How ?

  1. Set the version number you are going to document in the build.xml in <property name="version" value="X" />.
  2. Create a folder with the name of this version in the folder where the build.xml file lives.
  3. Copy the whole libraries folder inside the folder you created.
  4. Run the build script.

The resulting xml files will be places in a folder build.
